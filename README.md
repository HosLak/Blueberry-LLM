# Blueberry-AI
Train a GPT-4-level LLM on the whole internet for $100 by 2029

## Usage

1. Clone the repository:
```bash
git clone https://github.com/HosLak/Blueberry-LLM.git
```

2. Change into the project directory:
```
cd Blueberry-LLM
```

3. Run training:
python run_train.py

4. Run inference:
python inference.py
